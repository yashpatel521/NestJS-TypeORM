## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## APIs

```bash
# server status
$ http://localhost:5000/

# swagger
$ http://localhost:5000/swagger

# Add default users
$ http://localhost:5000/addDefaultUser

# Firebase Login page
$ http://localhost:5000/firebaseLogin

# Websockets
$ http://localhost:5000/webSockets
```
